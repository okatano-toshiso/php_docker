# LAMP
`PHPとWEBサーバ、mysql構築用のDockerファイル`

| apri |  name  | version  |
| :---: | :---: | :---: |
| lang | PHP | 7.4 |
| WEB | nginx | latest |
| DB |  mysql | latest  |

## START
### BUILD
```
docker-compose build
```
### UP
```
docker-compose up
```
### DOWN
```
docker-compose down
```
### STATUS
```
docker-compose ps
```
